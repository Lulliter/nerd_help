NOTE: If AWS did not automatically create a hosted zone upon buying the domain name, create one.

Tip

So for example if you bought the domain name username.com, you create the hosted zone name username.com. Once created open it up. Once you have created the hosted zone, open it up, click the Type NS, and copy the values on the right (4x url‚Äôs to awsdns addresses). Then go to the registered domains from the left menu, and in the Name Servers section on the right, click Add or edit name servers, delete any contents in there, and add the 4x lines you just copied one by one. This gives your domain name the address to the hosted zone.

[on AWS]: Configuring a subdomain www.example.com

  1. Create another A type of record set (your www.example.com rule, aka an alias for example.com)
